# 🏴‍☠️ Asutosh Panigrahi — Interactive Portfolio

A modern portfolio that showcases two sides of the same creator:

🎨 **UI/UX Designer**
💻 **Frontend Developer**

Built as an immersive experience rather than a static portfolio, featuring dynamic role switching, animated interactions, project showcases, and a visual journey map that tells the story behind the work.

---

## ✨ Features

### 🎭 Dual-Mode Experience

Switch seamlessly between:

* **Designer Mode** — focused on UX, research, wireframing, prototyping, and design projects.
* **Developer Mode** — focused on frontend development, Salesforce, React, and engineering projects.

Each mode dynamically updates:

* Hero section
* About section
* Skills
* Projects
* Resume downloads
* Visual elements

---

### 🏴‍☠️ Interactive Journey Map

A custom-built pirate-themed life timeline visualized as an interactive voyage.

Highlights include:

* Education milestones
* Certifications
* Internships
* Projects
* Career achievements

Every milestone represents a chapter of the journey from student to designer and developer.
